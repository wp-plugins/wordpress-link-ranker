== Description ==

This plugin adds the Google Page Rank and Alexa Rank to Links Page in WordPress admin panel. you can enable or disable them in the options page.
notice that if you have a lot of links in your Links page, it may take a long time to calculate the ranks.



* Plugin Sponsored by - [pcday.ir](http://www.pcday.ir)



== Installation ==

1. Upload `wp-links-ranker` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
